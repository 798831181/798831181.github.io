#### Aspect

切面


#### JoinPoint

连接点是程序执行过程中的一个点，例如：方法的执行、异常的捕获。

在springAOP中连接点通常是指方法的执行。


#### PointCut

切入点是一个断言，帮助与匹配一个通知，在特定的连接点上应用切面。


#### Advise 

通知是切面在特定连接点上的动作，包括前置通知、环绕通知和后置通知。
